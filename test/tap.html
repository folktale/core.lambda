This will run the tests in a Node environment, and present the results with the TAP reporter.

hifive = require 'hifive' tap = require 'hifive-tap' specs = require './specs'

If we, for any reason, fail any of the tests, signal that with an error exit status.

(hifive.run specs, tap!).otherwise -> process?.exit 1
